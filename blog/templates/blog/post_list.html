Steps to push new HTML to Git from Console

  • Type git add --all .
  • Type git commit -m "Insert Reason Here"
  • Type git push

Steps to pull new HTML from Git to PythonAnywhere

  • Go to PythonAnywhere and open "Consoles"
  • Open Bash
  • Ensure you're in the correct directory
  • Use cd ~/.pythonanywhere.com to cd if needed
  • Type git pull
  • Go to Web in PythonAnywhere and click Reload
